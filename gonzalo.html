git config --global user.name "" : Configure the author name and email address to be used with your commits.
git reset HEAD : reset current HEAD to the specified state.
gitk -all : is a command line utility to run git command under multiple git repositories concurrently.
git branch --merged : List, create, or delete branches.With --merged, only branches merged into the named commit.
git branch "branch" "commit hash" : List, create, or delete branches. Create branch of the chosen "commit hash".
git pull "remote" "branch" : Fetch from and integrate with another repository or a local branch.
